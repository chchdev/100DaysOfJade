# HelloWorld

A simple Hello World program written in [Jade Programming Language](https://www.jadeplatform.com/developer-centre).

## Overview

This project demonstrates basic Jade syntax and functionality with a minimal HelloWorld example that asks a user for their name and then displays the output to the user. 


## What I learned? 

- How to get user input and store them in variables. 
- How schemas work
- How to write basic methods
- How string concatenation works within Jade. 
- How to compile and execute programs in Jade and export them.
